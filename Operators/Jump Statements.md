# Jump Statements

  * break: exits a loop
  * continue: moves to the next iteration
  * goto: moves to a name block of code:

  

int x = 3;

if (x == 3)

     goto customLabel;

x++

  

customLabel:

Console.WriteLine(x);

//Displays 3
